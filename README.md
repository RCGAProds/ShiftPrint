# ShiftPrint

**Transform shifts into shareable schedules.**

ShiftPrint is a lightweight web application that converts raw work schedules into clean, customizable, and professional-looking timetables. Simply paste your shift data, customize the appearance, and export the final result as a high-quality image ready to share or print.

## Features

- 📋 Parse and organize raw shift schedules
- ⏱️ Automatically calculate working hours
- ☕ Calculate break times and rest periods
- 📅 Highlight working days and days off
- 🎨 Customize colors, fonts, and visual styles
- 🖼️ Export schedules as high-quality PNG images
- ⚡ Fast, simple, and fully browser-based
- 📱 Responsive design for desktop and mobile devices

## Use Cases

- Employees who want a cleaner view of their work schedule
- Shift workers who need to share their timetable with family or colleagues
- Teams looking for a printable version of their roster
- Anyone who prefers a visual and organized schedule format

## How It Works

1. Copy your shift schedule from your company portal or source.
2. Paste the schedule into ShiftPrint.
3. Review the automatically generated timetable.
4. Customize the appearance to your preference.
5. Export the result as a PNG image.

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Installation

Clone the repository:

```bash
git clone https://github.com/RCGAProds/shiftprint.git
```

Open the project folder:

```bash
cd shiftprint
```

Launch `index.html` in your browser or serve the project using a local web server.
